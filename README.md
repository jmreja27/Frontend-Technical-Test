# Steps to run application with commands
 1. Clone the repository
 2. Run 
 ```bash
 npm install
 ```
 3. To deploy application with Vite run
 ```bash 
 npm run dev 
 ```
 4. To run tests run
 ```bash 
 npm run test 
 ```
